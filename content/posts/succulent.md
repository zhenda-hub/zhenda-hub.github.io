## 通用多肉养护口诀（很重要）

> 多晒、少水、通风好，
> 土干再浇不出事。

### 土壤

- 一定要颗粒土（多肉土、赤玉土、珍珠岩）
- 不能用普通园土

### 浇水判断

- 手捏盆很轻 → 可以浇
- 叶子发软 → 可以浇
- 叶子饱满 → 别浇




## 知识

### 蒸腾

- 叶片的气孔出汗，吸收CO₂。 
- 是水，养分，矿物质的运输系统, 
- 给植物降温。


| 档位 | 状态         |
| -- | ---------- |
| 太小 | 水和 CO₂ 上不来 |
| 合适 | 光合效率最高     |
| 太大 | 脱水 → 气孔关   |


通风促进蒸腾

### 光合作用

$$6CO₂ + 6H₂O + 光能  →  C₆H₁₂O₆ + 6O₂$$



光合的作用：

生长，变胖，上色，修复伤口


- 不是越晒越好
- 高温下光合反而会下降


## 新手常见“养死原因”

- 浇水太勤（No.1 杀手）
- 室内不通风
- 光线太暗（只放书桌不靠窗）


### 常见不良症状

| 现象    | 含义       |
| ----- | -------- |
| 底叶干瘪  | 正常 / 适应  |
| 中心叶发软 | ⚠️ 缺水或根弱 |
| 整株塌   | ❌ 烂根     |
| 发黑发臭  | ❌ 立刻处理   |



## 四季规律

冬天, 活着就合格：

- 室内 ≥ 10℃
- 极少浇水
- 不通冷风
- 接受变丑

夏天养多肉：少水、通风、躲太阳
春秋：光照越多越好， 10天浇水一次
