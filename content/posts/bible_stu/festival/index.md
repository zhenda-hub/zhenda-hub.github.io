## 主要的基督教节日

| 节日名称               | 日期 (公历)            | 宗教意义                                                             |
| ---------------------- | ---------------------- | -------------------------------------------------------------------- |
| 圣诞节 (Christmas)     | 12月25日               | 纪念耶稣基督降生。这是全球影响范围最广的基督教节日。                 |
| 复活节 (Easter)        | 春分月圆后的首个星期日 | 纪念耶稣基督死后第三天复活。这是基督教最神圣的节日，象征希望与重生。 |
| 圣灵降临节 (Pentecost) | 复活节后第50天（周日） | 纪念圣灵降临在使徒身上，标志着基督教会的正式建立。                   |

## 受难周 (Holy Week)： 复活节前的一周

| 节日            | 中文名                 | 日期（2026年） | 纪念内容                                   |
| :-------------- | :--------------------- | :------------- | :----------------------------------------- |
| Palm Sunday     | 棕枝主日               | 3月29日        | 耶稣骑驴进入耶路撒冷，群众以棕榈枝铺路迎接 |
| Maundy Thursday | 濯足星期四(设立圣餐日) | 4月2日         | 最后的晚餐，耶稣为门徒洗脚，设立圣餐       |
| Good Friday     | 受难节（耶稣受难日）   | 4月3日         | 耶稣被钉十字架受死                         |
| Holy Saturday   | 圣周六                 | 4月4日         | 耶稣躺在坟墓中，教会守夜默哀               |
| Easter Sunday   | 复活节                 | 4月5日         | 耶稣从死里复活，是基督教最核心的节日       |
| Easter Monday   | 复活节星期一           | 4月6日         | 复活节次日，许多国家为公众假期             |
