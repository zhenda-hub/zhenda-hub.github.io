## vscode快捷键

| 功能                  | 快捷键             | 类型 |
| --------------------- | ------------------ | ---- |
| 导航                  | alt ,<-            | 查看 |
|                       | alt ,->            |      |
| 递归折叠全文          | **ctrl, k，0**     |      |
|                       | ctrl, k，j         |      |
| 递归折叠              | ctrl, k， [        |      |
|                       | **ctrl, k， ]**    |      |
| 折叠                  | ctrl, shift, [     |      |
|                       | ctrl, shift, ]     |      |
| 参数提示              | ctrl，shift，space |      |
| 搜索时,查看下一个变量 | ctrl, f3           |      |
| 换行                  | ctrl,enter         | 编辑 |
| 移动代码              | alt ,上箭头        |      |
|                       | alt ,下箭头        |      |
| 复制一行代码          | shift,alt,下       |      |
| 规范化代码            | alt, shift, f      |      |
| 多行编辑              | shift alt 拖动     |      |
